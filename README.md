# Designing and Compromising a Custom Active Directory Lab

## Overview
This project documents the design, deployment, and compromise of a
custom-built enterprise Active Directory environment.

The goal is to simulate a realistic **internal penetration test**

This is not a prebuilt lab or walkthrough-based exercise.

## Project Focus
- Enterprise Active Directory design
- Identity and access misconfigurations
- Kerberos and service account risks
- Group Policy abuse
- Privilege escalation modeling
- Professional security reporting

---

## Scope
- Internal network only
- Domain-joined Windows systems
- Assumed low-privileged domain user access
- No social engineering or external attacks
