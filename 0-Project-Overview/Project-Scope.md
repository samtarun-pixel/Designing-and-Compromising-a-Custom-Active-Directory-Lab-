# Project Scope

This project focuses on designing and compromising a
custom Active Directory lab to understand real-world
enterprise misconfigurations.

## Objectives
- Build a realistic enterprise AD environment
- Engineer common security weaknesses
- Assess internal attack paths
- Document findings professionally

## In Scope
- Domain Controller
- Employee workstation
- Domain users and service accounts
- Group Policy Objects

## Out of Scope
- Internet-facing services
- Physical attacks
- Social engineering
